About

snmpWalker is a webapp that allows you to perform a SNMP WALK or GET over a certain oid and prompts the result in human readable data. Not only in number format.

It is a full stack javascript app tested on a debian server with snmpd.

This web app by @_jesusdario is a project for Net Management at the University of Sevilla. Checkout the GitHub repo here . You can mail me! jesusdriru gmail.com.
